## GPU対応表
NVENCでの映像配信には対応したGPUが必要です。<br>ホストアプリ実行時に「**[ERR] nvEncOpenEncodeSessionEx failed: NV_ENC_ERR_NO_ENCODE_DEVICE (1)**」のログが表示された場合はGPUが対応していません。<br>将来CPUエンコーダーによる映像配信についても実装予定のためリリースをお待ちください。

| ---- |
| GTX 9xx |
| GTX 10xx |
| GTX 16xx |
| GTX 20xx |
| GTX 30xx |
| GTX 40xx |

## FAQ
### Q
PlayStationのコントローラーを使っているけど、ゲーム内のコントローラ表記がXboxになっている
### A
ゲームによってはゲーム表記がXboxになります

---

### Q
ホスト/クライアントからしかコントローラー入力ができない
### A
起動しているゲームが使用しているコントローラーに対応しているか確認してください。<br>ゲームによっては1つのコントローラーからしか入力を受け付けないものがあります<br>ゲームを一度終了し、クライアント・ホストのどちらかのコントローラーのみ電源を入れた状態でゲームを起動してみてください
